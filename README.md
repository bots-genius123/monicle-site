# monicle.ai — static website

The Monicle marketing site as a self-contained static site (HTML/CSS/JS). Migrated off
WordPress + Elementor so it can be edited in Git and deployed anywhere, with no database,
no plugins, and no server-side code to maintain or secure.

## Pages

The live site is 9 pages (English). Each is a folder with an `index.html`:

| URL | Folder | Nav label |
|---|---|---|
| `/` | `index.html` | Home |
| `/seo-service-detail-page/` | `seo-service-detail-page/` | Services |
| `/way-of-working/` | `way-of-working/` | Way Of Working |
| `/client-portfolio/` | `client-portfolio/` | Our Work |

## Structure

```
index.html              Home
<page>/index.html        Each inner page
assets/fonts/            Self-hosted web fonts (Plus Jakarta Sans, Inter, Material Icons)
wp-content/              CSS, JS, images, originally generated by Elementor (kept as-is)
robots.txt, *-sitemap.*  SEO files
```

The design CSS lives in `wp-content/uploads/elementor/css/` (per-page) and
`wp-content/uploads/elementor/css/custom-frontend.min.css` (global). Images are in
`wp-content/uploads/`. These paths are kept exactly as exported so the design renders
unchanged.

## Preview locally

```bash
python3 -m http.server 8099
# open http://localhost:8099
```

## Fonts

Fonts are self-hosted under `assets/fonts/` (not loaded from Google) — faster and
GDPR-compliant for EU visitors. Each page links `/assets/fonts/{plus-jakarta,inter,material-icons}.css`.
